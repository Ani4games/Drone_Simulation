# Drone vs Enemy Combat Simulation (Unity)

## Overview
A Unity prototype built as part of a Drone Combat Skill Assessment.
The focus is on gameplay logic and AI behavior rather than visuals.

## Controls
- W A S D – Move drone
- Q / E – Ascend / Descend
- Space – Fire missile

## Features
- Player-controlled drone
- Missile firing with explosion effects
- Enemy patrol using NavMesh
- Enemy detection and attack behavior
- Destroyable targets

## Tech Stack
- Unity 2022+
- C#
- NavMesh AI
- Rigidbody physics

## Known Limitations
- Basic enemy AI
- Placeholder models and effects
